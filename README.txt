Для того что бы запустить проект на локальной машине откройте папку под названием "сайт" - /main.html - двойной клик.
Либо откройте проект в Visual Studio Code. Так же выберите /main.html, Затем F5, выбираете браузер.
При наличии плагина Live Server проект так же можно запустить через него.
